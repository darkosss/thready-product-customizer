A WooCommerce extension that allows customers to customize products with print designs.

== Description ==
Thready Product Customizer allows you to add print designs to your WooCommerce products. 
Each product variation can have its own customized print position and size.

== Features ==
- Add print designs to product variations
- Customize print position and size for each variation
- Automatic image generation on save
- WebP format support with PNG fallback
- Optimized for performance

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/thready-product-customizer` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure your product print settings in the product edit screen
